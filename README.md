# Teste Gpsoft

## Instalação das Dependências

Para instalar as dependências do projeto Backend, siga os seguintes passos:

```bash
npm install
npm install knex
npm install nodemon

```

Para rodar o backend com nodemon, utilize:

```bash
npm run dev

```
